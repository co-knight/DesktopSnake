# 桌面贪吃蛇 (C/WinAPI)

**注意，该游戏只对 `Windows 10` 有效。** 因为微软在 `Windows 11` 修改了桌面API。

特色:
- 使用现有的桌面图标实现有效。
- 自动检测桌面图标网格大小。
- 禁用和重新启用“Snap to Grid”以实现平滑动画。
- 防止180度转弯。

控制:
- W / 上箭头:   向上
- S / 下箭头:   向下
- A / 左箭头:   向左
- D / 右箭头:  向右
- ESC:              退出游戏

如何编译（使用提供的Makefile）:
```
make
```
